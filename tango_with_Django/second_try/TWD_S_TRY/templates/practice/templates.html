Filters

  • | upper (:) >>>> {{context.name | upper}}

  • text | escape | linebreaks (:) combierte el texto en parrafos
    • >>>> {{context.parrafo|escape|linebreaks}}
  • bio|truncatewords:30 (:) This will display the first 30 words of the bio variable.
    • >>>> {{context.parrafo|truncatewords:15}}

  • list|join:", " (:) join con cada elemento de la lista
    • >>>> {{context.lista | join:", "}}

  • value|default:"nothing" (:) If a variable is false or empty, use given default. Otherwise, use the value of the variable
    • >>>> {{context.value | default:'variable_vacia'}}

  • value|length (:) Returns the length of the value. This works for both strings and lists. For example:
    • >>>> length de lista: {{context.lista | length}}
